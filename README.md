# Project Setup

## Install Dependencies Backend

To install the necessary dependencies, run:

```bash
pip3 install -r requirements.txt
```

Run Server

```bash
   python app.py
```
## Install Dependencies Frontend

To install the necessary dependencies, run:

```bash
npm install
```
Run Server

```bash
   npm start
```
